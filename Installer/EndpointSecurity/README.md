# Endpoint Security installer notes

This folder holds **example** material for targets that will run a live Endpoint Security client (Xcode app or system extension). It does not perform installation by itself.

- `entitlements.example.plist` — copy into your Xcode target and rename; enable the capability in the Developer portal.

Full steps: [docs/endpoint-security-xcode-integration.md](../../docs/endpoint-security-xcode-integration.md).
