# Service Registration Scaffold

This directory is the Phase 2 placeholder for the future helper install/runtime path.

Current intent:

- package install-plan resources with the beta app
- define the eventual system extension or helper registration artefacts
- give the app a concrete way to report that install scaffolding exists even though installation is not implemented yet

This directory does **not** mean the helper is installable today.

The remaining work is still:

- Apple-granted Endpoint Security entitlement approval
- a real system extension or helper registration target
- user approval and lifecycle management
- app-driven install/start/stop/status flows
